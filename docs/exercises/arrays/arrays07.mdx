Erstelle eine ausführbare Klasse, welche zwei Zahlenfelder zu einem Zahlenfeld
zusammenführt und alle Elemente des zusammengefügten Zahlenfeldes auf der
Konsole ausgibt.

## Zahlenfelder

```java
int[] arrayOne = { 1, 2, 3, 4 };
int[] arrayTwo = { 5, 6, 7, 8 };
```

## Konsolenausgabe

```console
1
2
3
4
5
6
7
8
```
