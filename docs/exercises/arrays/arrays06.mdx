Erstelle eine ausführbare Klasse, welche ein gegebenes mehrdimensionales
Zahlenfeld analysiert.

```java
int[][] multiDimArray = {
    { 5, 8, 2, 7 },
    { 9, 6, 10, 8 },
    { 10, 2, 7, 5 },
    { 1, 9, 5, 4 } };
```

Es soll jeweils der kleinste sowie der größte Wert einer Reihe auf der Konsole
ausgegeben werden.

## Konsolenausgabe

```console
MIN - MAX
2 - 8
6 - 10
2 - 10
1 - 9
```
