Berechne die Hashwerte der Objekte `Person[name=Petra, age=33, gender=w]`,
`Person[name=Heidi, age=17, gender=w]` und `Person[name=Max, age=47, gender=m]`
gemäß der Formel `Hashwert Name / Hashwert Alter + Hashwert Geschlecht`.

  </TabItem>

(80 + 101 + 116 + 114 + 97) / 33 + 119 = 134

  </TabItem>
  <TabItem value="c" label="Hashwert 2">

(72 + 101 + 105 + 100 + 105) / 17 + 119 = 147

  </TabItem>
  <TabItem value="d" label="Hashwert 3">
