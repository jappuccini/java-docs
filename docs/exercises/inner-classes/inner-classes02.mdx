- Passe die Klassen `Company` und `Employee` aus Übungsaufgabe
  [Exceptions03](../exceptions/exceptions03) anhand des abgebildeten
  Klassendiagramms an
- Passe die ausführbare Klasse aus Übungsaufgabe
  [Exceptions03](../exceptions/exceptions03) so an, dass sie fehlerfrei
  ausgeführt werden kann
- Die Klasse Emplyee soll hierbei eine Element Klasse sein.

## Klassendiagramm

```mermaid
classDiagram
    Company *-- Employee
    Employee o-- Person

    class Company {
        -name: String &#123final&#125
        -employees: List~Employee~ &#123final&#125
        -numberOfEmployees: int
        +Company(name: String)
        +name() String
        +employees() List~Employee~
        +getNumberOfEmployees() int
        +addEmployee(employee: Employee) void
        +print() void
    }

    class Person {
        -name: String &#123final&#125
        +Person(name: String)
        +name() String
    }

    class Employee {
        -employeeId: int &#123final&#125
        -person: Person &#123final&#125
        -salary: int
        +Employee(employeeId: int, person: Person, salary: int)
        +employeeId() int
        +name() String
        +setSalary(salary: int) void
        +getSalary() int
        +print() void
    }
```
