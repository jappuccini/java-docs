- Erstelle die Klassen `Company`, `Employee` und `Person` anhand des
  abgebildeten Klassendiagramms
- Erstelle eine ausführbare Klasse, welche ein Unternehmen mit mehreren
  Mitarbeitern erzeugt und auf der Konsole ausgibt

## Klassendiagramm

```mermaid
classDiagram
    Company o-- Employee
    Employee o-- Person

    class Company {
        -name: String &#123final&#125
        -employees: ArrayList~Employee~ &#123final&#125
        -numberOfEmployees: int
        +Company(name: String)
        +getName() String
        +getEmployees() ArrayList~Employee~
        +getNumberOfEmployees() int
        +addEmployee(employee: Employee) void
        +toString() String
    }

    class Person {
        -name: String &#123final&#125
        +Person(name: String)
        +getName() String
    }

    class Employee {
        -employeeId: int &#123final&#125
        -person: Person &#123final&#125
        -salaryInEuro: int
        +Employee(employeeId: int, person: Person, salaryInEuro: int)
        +getEmployeeId() int
        +getName() String
        +setSalaryInEuro(salaryInEuro: int) void
        +getSalaryInEuro() int
        +toString() String
    }
```

## Hinweis zur Klasse _Person_

- Der Konstruktor soll alle Attribute initialisieren
- Die Methode **getName** soll den Namen der Person zurückgeben

## Hinweise zur Klasse _Employee_

- Der Konstruktor soll alle Attribute initialisieren
- Die Methode **getEmployeeId** soll die Id des Mitarbeiters zurückgeben
- Die Methode **getName** soll den Namen des Mitarbeiters zurückgeben
- Die Methode **setSalaryInEuro** soll das Gehalt des Mitarbeiters festlegen.
- Die Methode **getSalaryInEuro** soll das Gehalt des Mitarbeiters zurückgeben.
- Die Methode **toString** soll alle Attribute des Objektes als String
  zurückgeben.

  Format: `Mitarbeiternummer - Name - Gehalt`

  Beispiel: `4001 - Steffen - 40000€`

## Hinweise zur Klasse _Company_

- Der Konstruktor soll alle Attribute initialisieren
- Die Methode **getName** soll den Namen der Firma zurückgeben
- Die Methode **getEmployees** soll die Mitarbeiterliste zurückgeben
- Die Methode **getNumberOfEmployees** soll die Anzahl der Mitarbeiter
  zurückgeben
- Die Methode **addEmployee** soll den eingehenden Mitarbeiter der
  Mitarbeiterliste hinzufügen
- Die Methode **toString** soll alle Attribute des Objektes und alle Mitarbeiter
  als String zurückgeben.

  Format:

  ```console
  Firmenname (Mitarbeiteranzahl)
  Mitarbeiternummer - Name - Gehalt
  ```

  Beispiel:

  ```console
  Softwareklitsche GmbH (3 Mitarbeiter)
  4001 - Steffen - 40000€
  4002 - Marianna - 120000€
  4003 - Mirco - 100000€
  ```

:::note Hinweis

Um in einem String einen Zeilenumbruch zu erzeugen muss `\n` verwendet werden.
Der String `Hello World` wird ohne Umbruch in einer Zeile dargestellt. Der
String `Hello\nWorld` wird mit Umbruch in zwei Zeilen dargestellt.

:::
