Erstelle eine ausführbare Klasse, welche den Wochentag sowie die Anzahl Tage bis
Weihnachten eines eingegebenen Datums ermittelt und ausgibt.

## Konsolenausgabe

```console
Gib bitte ein Datum ein (dd.mm.yyyy): 04.01.2016
Wochentag: MONDAY
Tage bis Weihnachten: 355
```

## Hinweis

Die Klasse `String` stellt für das Auslesen einer Teilzeichenkette eine passende
Methode zur Verfügung.
