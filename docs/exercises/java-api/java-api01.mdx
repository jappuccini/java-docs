Erstelle eine ausführbare Klasse, welche den Sinus von 0.0 bis 1.0 in
Zehnerschritten tabellarisch auf der Konsole ausgibt.

## Konsolenausgabe

```console
x = 0.0, sin(x) = 0.0
x = 0.1, sin(x) = 0.1
x = 0.2, sin(x) = 0.2
x = 0.3, sin(x) = 0.3
x = 0.4, sin(x) = 0.4
x = 0.5, sin(x) = 0.5
x = 0.6, sin(x) = 0.6
x = 0.7, sin(x) = 0.6
x = 0.8, sin(x) = 0.7
x = 0.9, sin(x) = 0.8
x = 1.0, sin(x) = 0.8
```

## Hinweis

Die Klasse `Math` stellt für die Sinus-Berechnung eine passende Methode zur
Verfügung.
