  4-5-6-Würfeln oder 1-2-3-Würfeln zu würfeln

## Klassendiagramm

```mermaid
classDiagram
    DiceGame o-- Dice
    DiceGame o-- Player
    DiceGame o-- DiceCup
    Dice <|-- LowValueDice
    Dice <|-- HighValueDice

    class Player {
        -name String
        -score int
        -isActive boolean
        -specialAvailable boolean
        +Player(name String)
        +getName() String
        +getScore() int
        +setScore(int score) void
        +isActive() boolean
        +setActive(isActive boolean) void
        +getSpecialAvailable() boolean
        +setSpecialAvailable(specialAvailable boolean) void
    }

    class Dice {
        -id int
        #value int
        +Dice(id int)
        +getId() int
        +getValue() int
        +rollTheDice() void
    }

    class LowValueDice {
        +LowValueDice(id int)
        +rollTheDice() void
    }

    class HighValueDice {
        +HighValueDice(id int)
        +rollTheDice() void
    }

    class DiceCup {
        +rollTheDices(dices Dice...) void
    }

    class DiceGame {
        -diceCup DiceCup
        -dices ArrayList~Dice~
        -players ArrayList~Player~
        -scanner Scanner
        +DiceGame(players ArrayList~Player~)
        +start() void
        +move(player Player) void
    }
```

## Hinweis zur Klasse HighValueDice

Die Methode `void rollTheDice()` soll nur 4er, 5er und 6er "würfeln".

## Hinweis zur Klasse LowValueDice

Die Methode `void rollTheDice()` soll nur 1er, 2er und 3er "würfeln".
